# i2Message 0.1.7

This release makes sent-message editing clearer, faster, and more native.

- Redesigns the edit sheet around the original message and its replacement.
- Adds focused, empty, unchanged, ready, and expired interaction states.
- Shows a live edit-window countdown and the number of edits remaining.
- Explains the secure Messages handoff as a compact three-step sequence.
- Uses Command-Return to continue so Return remains available for multiline edits.
- Improves VoiceOver labels, keyboard hints, loading feedback, and light/dark appearance.
