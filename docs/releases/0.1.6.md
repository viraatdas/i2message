# i2Message 0.1.6

This release adds a safe sent-message text editing workflow.

- Adds **Edit Text…** to the action pill for outgoing text messages.
- Provides a focused replacement-text editor with unchanged/empty validation.
- Enforces Apple's supported-conversation, five-edits, 15-minute rules for live messages, including mixed-protocol groups with an iMessage participant.
- Copies live replacement text and opens Messages.app with explicit native completion steps; i2Message never writes private Messages storage.
- Updates fixture messages locally and preserves their previous text in edit history.
- Adds regression coverage for local edits, unsupported messages, edit limits, and the live paste/open handoff.
