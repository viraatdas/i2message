# i2Message 0.1.8

This release fixes thread avatars and keeps conversation navigation responsive while local search indexes rebuild.

- Shows the current user's macOS Me-card photo for outgoing thread replies.
- Replaces ambiguous question-mark avatars with a neutral person fallback.
- Uses the same sender avatar in reply chips and the expanded thread panel.
- Isolates background indexing from interactive Messages database reads.
- Runs indexing at background priority and reduces progress-driven UI refreshes.
- Caches decoded contact photos to remove repeated image decoding while scrolling.
