# i2Message 0.1.5

This release repairs stale contact metadata in local search results.

- Refreshes message and attachment search records when conversation titles, sender names, handles, services, or contact IDs change.
- Forces one safe, resumable index-format upgrade so records created before Contacts access are corrected.
- Restarts background indexing immediately after Contacts or Full Disk Access becomes available.
- Preserves incremental indexing and existing semantic embeddings when the indexed data is unchanged.
